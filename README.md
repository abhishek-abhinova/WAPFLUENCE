# Wapfluence - WhatsApp Influencer Marketing Marketplace

## Project Structure

```
WAPFLUENCE/
├── index.html              # Landing page
├── public/
│   ├── css/
│   │   └── style.css       # Main stylesheet
│   ├── js/
│   │   └── main.js         # Main JavaScript
│   └── images/             # Images and assets
├── pages/
│   ├── influencer/         # Influencer portal pages
│   ├── business/           # Business portal pages
```

## Getting Started

1. Open `index.html` in your browser
2. Navigate through different portals using the links
